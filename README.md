# **Evil Bear Grill & Bar**

A brutal and atmospheric grill & bar website built as part of [The Odin Project's Restaurant Page curriculum](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page). The entire UI of this single-page application is dynamically generated using vanilla JavaScript, modern DOM manipulation, and ES6 modules bundled with Webpack.

## **🚀 Quick Start & Build**

This project is managed and compiled using the **npm** package manager.

### **1\. Install Dependencies**

npm install

### **2\. Run Development Server**

npm start

### **3\. Build for Production**

npm run build

The compiled, production-ready assets will be generated in the dist/ directory, structured into js/, css/, and img/ folders alongside a clean index.html.
